# 📸 地下偶像摄影指南

> 想在地偶演出时为你的推拍摄精彩场照？查收这份指南！
>

🌟 地下偶像摄影的难点：

1. **环境昏暗**——拍出来的照片不够亮
2. **XOX动作太快**——画面糊了
3. **光线诡异**——人脸很黑、肤色不对

本指南用于无痛解决上述问题，只要工具到位，摄影即是一项浅显技术，遵从本指南的固定套路，效果一定不会差 🚀

这个教程会以镜头群丰富的**索尼微单机型**作为示例。

---

## 🚀 太长不看版

+ **相机档位**：M档
+ **光圈**：开到最大
+ **快门速度**：1/500s 及以上
+ **ISO：** 自动
+ **对焦模式：** AF-C 连续对焦
+ **对焦区域：** 广域 or 跟踪扩展点
+ **测光模式**：多重测光 + 点测光
+ **连拍模式**：高\中速连拍
+ **拍摄格式**：RAW格式
+ **焦段推荐**：50mm定焦
+ **后期处理**：像素蛋糕 🍰

## 📚 详细介绍版

### 📈 1. 相机拍摄参数设置

#### 🎯 第一要点：高快门速度

地偶拍摄想要不糊片，快门速度一定要足够高，推荐至少 1/500s 以上，以获得主体清晰的照片

**示例1 ：** 以曝光时间 1/500s 拍摄——主体清晰（但手部区域略微模糊）：

![](images/1_1_500s.png)


****

**示例2：** 以曝光时间 1/800s拍摄，获得空中定格的清晰画面：

<img src="images/2_1_800s.png"  style="max-height: 600px; display: block; margin: auto;">

#### 🎯 第二要点：大光圈

地偶场地光线通常较为昏暗，1/500s的高快门速度进一步限制了进光量，为此 **大光圈镜头必不可少** 。定焦镜头推荐光圈值 F1.8 及以上，变焦镜头推荐
F2.8 及以上。

示例以 F1.4光圈、1/500s 拍摄，进光量不足导致ISO飙到了5000，图片在放大后（右侧）细节丢失：

![](images/3_f1.4_iso_5000.png)


#### 🌟感光度ISO：自动

当快门速度、光圈值确定后，照片的噪点水平也就固定了，ISO保持自动即可。

> 不要人为限制感光度，如果感觉整体画面过暗或过亮，可以调整“曝光补偿拨盘”，或者通过后期处理来调整，这两者的效果是一致的。


#### 🌟对焦模式：AF-C 连续对焦

AF-C 模式适用于拍摄运动对象，因此这个选项模式在地偶拍摄中是最佳选项。

#### 🌟对焦区域

+ **广域**

<img src="images/4_guangyu.png"  style="max-height: 200px; display: block; margin: auto;">


图省事可以选择广域模式，像手机一样直接拍即可，对焦位置可以戳屏幕调整。

+ **跟踪：扩展点**

<img src="images/5_genzong.png"  style="max-height: 200px; display: block; margin: auto;">


**“跟踪:扩展点”是最佳的对焦区域设置**，只不过有一点理解成本；它相较于“广域” 有两个优势：

    - 可以单手控制对焦位置（先通过半按快门锁定对焦位置，再移动相机进行构图），相较于戳屏幕更方便
    - 该模式支持调整“测光点位置”以实现精确测光，而“广域”不支持（下一节详述）

对“跟踪:
扩展点”的介绍推荐看UP主“阿里”的教学视频，预估学习成本10分钟：[https://www.bilibili.com/video/BV1ZZ4y1p7x7](https://www.bilibili.com/video/BV1ZZ4y1p7x7)

#### 🎯 测光模式

舞台灯光场景复杂多变，容易出现“大光比”场景，比如图示人物很黑、背景过曝 😭：

<img src="images/6_guangbi.png"  style="max-height: 600px; display: block; margin: auto;">



针对这个问题的思路是：**优先保证XOX面部曝光正确**。 在地偶拍摄中用以下两个测光模式即可：

1. **多重测光**

<img src="images/7_ceguang_duochong.png"  style="max-height: 200px; display: block; margin: auto;">

多重测光是相机最智能的测光模式，它基于整个画面的亮度分布计算出合适的曝光值。应该**优先选择多重测光模式**
，即“先相信相机的判断”。此外索尼相机记得打开“[多重测光人脸优先](https://helpguide.sony.net/ilc/2360/v1/zh-cn/contents/0406M_face_prty_in_mlti_mtr.html?search=%E5%A4%9A%E9%87%8D%E6%B5%8B%E5%85%89)
”设置，它会进一步保证人脸正确曝光。

<img src="images/8.png"  style="max-height: 200px; display: block; margin: auto;">

2. **点测光**

<img src="images/9.png"  style="max-height: 200px; display: block; margin: auto;">

**当光线环境太复杂、多重测光不好使时，应该将测光模式更换为“点测光”**。点测光
只对画面中的一个小圆形区域进行测光，拍摄时将测光点对准XOX面部，可以让面部获得最精确的曝光。

索尼相机的测光点位置默认处于屏幕中心，可以进一步调整测光点位置以更好地构图：


<img src="images/11.png"  style="max-height: 350px; display: block; margin: auto;">


**测光点位置调整方法：**

1. 确保“对焦区域” 设置为 “**跟踪: 扩展点**” （ “广域”对焦区域无法调整测光点位置）
2. 打开 点测光点 与 “对焦点联动” 选项

<img src="images/10.png"  style="max-height: 200px; display: block; margin: auto;">

3. 在屏幕上戳一下，选定了对焦位置的同时，也同时确定了测光点位置。

如果拍摄时祭出点测光依旧无能为力，那你可能已经触及人类科技的边缘，不妨放弃测光，直接发力后期修图吧。

### 🎯 2. 镜头与焦段选择

焦段选择与场地大小以及站位有关。

1. **提前了解场地**

提前了解舞台到距离观众的距离，一些小场地舞台距离可能非常近，常规35mm及以上焦段很难拍到全身。以经典的人像焦段为例，焦段与所需的最小拍摄距离如下（横置）：

+ **35mm**：距离 3m左右能拍到全身
+ **50mm**：距离 4m左右能拍到全身
+ **85mm**：距离 6m左右能拍到全身


2. **早点去排队**（对身高有自信忽略这条）

第一排站位更容易出片，但是要避开中间起飞区（如果介意自己飞起来︿(￣︶￣)︿）；一般推荐全画幅50mm焦段。如果站在第一排边缘位置，推荐
85mm定焦\长焦变焦镜头。

3. **了解XOX站位规律**

很多XOX会习惯性地看向舞台某一侧，仔细斟酌站在台下的哪一边，以获得更多正面视角。

+ 半画幅镜头评级：[https://sonyalpha.blog/2020/03/02/which-lenses-are-the-sharpest-for-a-a6xxx/](https://sonyalpha.blog/2020/03/02/which-lenses-are-the-sharpest-for-a-a6xxx/)

建议 **只考虑评级在 Outstanding 与 Excellent** 的镜头，地偶拍摄很容易画质崩坏，先把硬件搞到位。

#### 🎯 机型推荐

##### 🎯 索尼全画幅

+ **机身**：
    - A7C：二手价格不到7000，直出色彩问题可以通过后期搞定。
    - A7C2：3300万像素，水桶机。
    - A7R4/A7CR/A7R5：6100万像素，后期裁切空间足，方便构图。
+ **镜头**：
    - 索尼 24\35\50\85\135 GM系列
    - 适马 85mm F1.4：Outstanding评级，以及来自"85定焦镜头专家"的认证
    - 索尼70-200 GM2：Outstanding评级，掏出这个你才是名副其实的炮哥...
    - 腾龙70-180 F2.8 一代：Excellent评级，二手价格不到4000，相较70-200 GM2 性价比溢出

---

##### 🎯 索尼半画幅

+ **机身**：

**仅推荐 A6400 与 ZVE-10 一代**，理由是高性价比，二手3500可淘到成色不错的机身。而更贵的机型，如“ZVE-10二代”缺少机械快门，拍照不如一代；半画幅机旗舰
A6700 价格超越了全画幅A7C，从地偶拍摄效果而言，不如换全画幅。

+ **镜头**：
    - 唯卓仕 27mm F1.2 Pro：Outstanding评级，等效全画幅 40mm。
    - 适马 56mm F1.4：Outstanding评级，等效全画幅 84mm。
    - 适马 18-50mm F2.8 DC DN：Excellent 评级，小巧轻，便高锐度。

### 📷 3. 实拍与后期

#### 🎯 先干他个一万张 💥

相机一定要设置到“连拍模式”，相机写卡速度足够快就选**高速连拍**，写的慢可以选中\低速。

预估精彩瞬间，提前按死快门，相机没卡死前不要停！有海量的前期拍摄，一定能甄选出精彩瞬间，要的就是万里挑一。
